자주 묻는 질문

CloudPress에 대한 모든 궁금증을 해결하세요.

서비스 일반

CloudPress는 Cloudflare Pages를 기반으로 작동하며, 별도 VPS 관리 없이 자동 확장·DDoS 방어·글로벌 CDN이 기본 포함됩니다. 전통적인 LAMP 스택 대신 Cloudflare D1(SQLite)과 KV를 사용하여 더 빠르고 안정적입니다.

WordPress REST API(/wp-json/wp/v2/)와 완전히 동일한 엔드포인트 구조를 제공합니다. 글 작성·편집, 카테고리·태그, 미디어, 사용자 권한 관리 등 핵심 기능이 100% 호환됩니다. 기존 워드프레스 사용자라면 동일한 방식으로 사용 가능합니다.

버튼 하나 클릭 후 3~8분 내에 Cloudflare Pages 프로젝트, D1 데이터베이스, KV 스토리지 생성, 바인딩 설정, CMS 배포까지 모두 자동 완료됩니다. 사이트 DNS 전파는 최대 5분 추가 소요될 수 있습니다.

현재 무료 플랜은 제공하지 않습니다. 스타터·프로·엔터프라이즈 유료 플랜을 제공합니다. 자세한 가격은 요금제 페이지에서 확인하세요.

기술 및 인프라

197 Tbps 네트워크 용량 기반. L3/L4 및 L7 DDoS 공격을 모두 자동 탐지·차단하며, 추가 비용 없이 무제한 제공됩니다. Cloudflare Magic Transit이 공격을 밀리초 단위로 완화합니다.

각 사이트마다 Cloudflare D1(SQLite 기반) 데이터베이스가 자동 생성됩니다. KV 스토리지는 세션·설정·캐시에 사용됩니다. 모든 데이터는 Cloudflare 인프라 내에 안전하게 저장됩니다.

네. 사이트 설정에서 커스텀 도메인을 입력하고, DNS에서 생성된 .pages.dev 도메인으로 CNAME 설정하면 됩니다. SSL 인증서는 자동 발급됩니다.

스타터 플랜은 7일, 프로 플랜은 30일, 엔터프라이즈는 실시간 백업을 제공합니다. D1 데이터베이스와 KV 스토리지 모두 백업됩니다.

결제 및 플랜

언제든지 업·다운그레이드 가능합니다. 업그레이드는 즉시 적용되며, 잔여 금액은 크레딧으로 제공됩니다.

토스페이먼츠를 통해 신용카드, 체크카드, 계좌이체 등 다양한 결제 수단을 지원합니다.

결제 후 7일 이내 환불 요청 시 전액 환불 가능합니다. 7일 이후에는 잔여 기간에 해당하는 금액을 일할 계산하여 환불합니다. 자세한 내용은 이용약관을 참조하세요.
