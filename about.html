CloudPress 소개

워드프레스의 호환성, Cloudflare의 인프라. 서버 없이 운영되는 새로운 방식의 CMS 호스팅을 만들고 있습니다.

우리의 미션

복잡한 서버 없이,
누구나 빠른 사이트를

CloudPress는 전통적인 LAMP 스택의 복잡함 없이, Cloudflare의 글로벌 엣지 네트워크 위에서 WordPress와 100% 호환되는 CMS를 운영할 수 있는 서비스입니다.

VPS 관리, 서버 패치, 보안 설정 없이 버튼 하나로 사이트를 만들고, 전 세계 어디서든 빠르게 접근 가능한 환경을 제공합니다.

개발자도, 비개발자도 — 누구나 프로 수준의 인프라를 손쉽게 활용할 수 있도록 만드는 것이 CloudPress의 목표입니다.

300+ Cloudflare 글로벌 데이터센터
197 Tbps DDoS 방어 네트워크 용량
3~8분 사이트 배포 완료 시간
100% WordPress REST API 호환
기술 스택

최신 엣지 컴퓨팅 기술로 구축

전통적인 서버 없이, Cloudflare의 분산 인프라를 100% 활용합니다.

⚡
Cloudflare Workers
서버리스 런타임. 전 세계 엣지에서 요청을 처리하여 지연 시간을 최소화합니다.
🗄️
Cloudflare D1
SQLite 기반 분산 데이터베이스. 관리형 DB로 별도 설정 없이 즉시 사용 가능합니다.
🗂️
Cloudflare KV
글로벌 키-값 스토리지. 세션, 캐시, 설정 데이터를 엣지에서 빠르게 처리합니다.
📄
Cloudflare Pages
정적 사이트 및 함수 호스팅 플랫폼. 무한 확장과 자동 배포를 지원합니다.
🛡️
Cloudflare WAF
L7 애플리케이션 방화벽. 악성 트래픽, 봇, DDoS를 자동으로 차단합니다.
🌐
WordPress REST API
/wp-json/wp/v2/ 완전 호환. 기존 WordPress 클라이언트와 플러그인을 그대로 사용합니다.
핵심 가치

왜 CloudPress인가

🚀

배포 속도 우선

복잡한 설정 없이 3~8분 만에 완전한 CMS 사이트를 자동 배포합니다. 개발자의 시간은 코드에 집중되어야 합니다.

🔒

보안은 기본

DDoS 방어, WAF, SSL 자동화가 모든 플랜에 기본 포함됩니다. 보안 설정에 시간을 낭비하지 마세요.

🌍

글로벌 성능

Cloudflare의 300+ 데이터센터에서 서빙됩니다. 한국이든 유럽이든, 빠른 응답 시간을 보장합니다.

🔗

완전한 호환성

WordPress REST API와 100% 동일한 구조. 기존 워드프레스 생태계의 도구와 클라이언트를 그대로 활용합니다.

"시간은 그 무엇보다 값진 "돈" 입니다. 서버를 관리하는 데 쓰는 돈이 아깝습니다. 콘텐츠와 비즈니스에 집중할 수 있도록, 인프라는 우리가 맡겠습니다."
CloudPress CEO 최성원
Cloudflare 기반 CMS 호스팅 전문
운영 철학

관리자 중심 운영

CloudPress는 셀프 서비스 결제 없이, 관리자가 각 사용자의 플랜과 사이트를 직접 관리하는 방식으로 운영됩니다.

👤

사용자

대시보드에서 사이트를 생성·관리하고 콘텐츠를 작성합니다. 플랜 변경이 필요하면 관리자에게 요청합니다.

🔧

관리자

모든 사용자의 플랜, 사이트, 공지를 직접 설정합니다. Cloudflare 인프라 설정도 관리자 페이지에서 통합 관리합니다.

⚡

자동 배포

Cloudflare Pages, D1, KV, 바인딩 설정이 모두 자동화됩니다. 클릭 한 번으로 완전한 사이트가 배포됩니다.

🌐

도메인 관리

Cloudflare DNS와 직접 연동되어 도메인 추가·DNS 레코드 관리가 대시보드에서 가능합니다.
