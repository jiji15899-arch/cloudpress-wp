CloudPress의 모든 기능

Cloudflare의 강력한 인프라 위에서, 워드프레스와 100% 동일한 경험을 제공합니다.

Cloudflare 글로벌 CDN
전 세계 300개 이상의 Cloudflare 엣지 서버에서 콘텐츠가 서빙됩니다. 방문자 위치와 가장 가까운 서버에서 응답하여 TTFB를 최대 70% 개선합니다.
  • 300개+ 글로벌 엣지 노드
  • Anycast 네트워크 라우팅
  • 스마트 캐싱 자동 적용
  • 이미지 자동 최적화 (WebP 변환)
🌍 글로벌 CDN 시각화
자동 DDoS 방어
197 Tbps 네트워크 용량의 Cloudflare Magic Transit이 L3/L4/L7 DDoS 공격을 자동 탐지·차단합니다. 추가 비용 없이 무제한 제공.
  • L3/L4/L7 멀티레이어 방어
  • 0ms 레이턴시 페널티
  • 197 Tbps 네트워크 용량
  • 실시간 위협 인텔리전스
🛡️ DDoS 방어 대시보드
워드프레스 100% 호환 CMS
WordPress REST API와 완전히 동일한 인터페이스. 글 작성, 카테고리, 태그, 미디어 업로드, 플러그인까지 워드프레스와 동일한 경험.
  • WordPress REST API 완전 호환
  • 블록 에디터 (Gutenberg 스타일)
  • 카테고리·태그·미디어 관리
  • 사용자·권한 관리
  • wp-json API 엔드포인트
📝 CMS 에디터 화면
원클릭 자동 구축
버튼 하나로 Cloudflare Pages 프로젝트, D1 데이터베이스, KV 스토리지, 바인딩 설정까지 자동으로 완료됩니다. 5분 안에 사이트 시작.
  • Pages 프로젝트 자동 생성
  • D1 DB + KV 자동 생성 및 바인딩
  • CMS 스키마 자동 초기화
  • 관리자 계정 자동 설정
  • SSL 자동 적용

  • Cloudflare WAF (Pro 이상)

  • Bot Fight Mode
