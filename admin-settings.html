🌐 WP Origin 설정 (핵심)

CloudPress가 WordPress 요청을 프록시할 단일 Origin 서버 정보를 입력합니다.
Worker는 이 Origin URL로 요청을 전달하며, 공유 시크릿으로 인증합니다.

WordPress가 실제로 실행 중인 Origin 서버의 전체 URL
Worker와 Origin mu-plugin이 공유하는 인증 시크릿
⚠️ 변경 후 Worker 재배포 필요
WP Origin URL 또는 시크릿 변경 시, wrangler.toml의 환경변수도 함께 업데이트하고 Worker를 재배포해야 합니다.
💳 플랜별 사이트 한도
⚡ Cloudflare CDN 설정

사이트 생성 시 D1/KV 리소스를 자동으로 생성하고 Worker Route를 등록합니다.

Workers, D1, KV 쓰기 권한이 있는 API 토큰
사용자 도메인 DNS CNAME 안내용 주소

⚙️ Worker Script 자동 업로드 바인딩 ID

wrangler.toml 의 database_id / KV namespace id 를 입력하세요.
