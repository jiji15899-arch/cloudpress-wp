🐙 GitHub CMS 소스 설정 (핵심)

사이트 생성 시 GitHub 레포에서 cloudflare-cms 코드를 HTTP fetch하여 Workers Script Upload API로 업로드합니다.
직접 파일 업로드가 아닌, 코드를 코드로 주입하는 방식입니다.

cloudflare-cms 소스가 있는 GitHub 레포. 예: myorg/cloudflare-cms
기본값: main
비공개 레포라면 repo 스코프 PAT 필요. 공개 레포면 비워도 됩니다.
💳 플랜별 사이트 한도
⚡ Cloudflare 설정

사이트 생성 시 D1/KV를 자동 생성하고 Workers Script Upload API로 CMS 코드를 업로드합니다. Worker Route와 DNS CNAME도 자동으로 등록됩니다.

Workers, D1, KV, DNS 쓰기 권한이 있는 API 토큰
cloudpress 플랫폼 자체 Worker 이름 (wrangler.toml의 name)
사용자 도메인 DNS CNAME 안내용 주소

⚙️ 메인 바인딩 UUID

비워두면 Pages 프로젝트에서 자동 탐색합니다. 탐색 실패 시 수동 입력하세요.
