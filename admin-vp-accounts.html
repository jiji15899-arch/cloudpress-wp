💡 VP 계정이란?

VP(Vesta Panel) 계정은 사용자들이 생성하는 WordPress 사이트가 배포될 호스팅 서버 계정입니다.
각 VP 계정마다 복제할 ZIP 템플릿 파일을 설정할 수 있으며, 사이트 생성 시 자동으로 해당 템플릿을 복제하여 새 사이트를 생성합니다.
각 사이트는 독립적인 D1 데이터베이스와 KV 네임스페이스를 사용하므로 완전히 격리된 환경에서 운영됩니다.
